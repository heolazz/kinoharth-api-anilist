Direct Text Search Endpoint

Use this route to search the AniList database for anime titles. The endpoint handles multi-word matching, fuzzy comparisons, and sorts based on match relevance and popularity.

URL Path Method Parameters Status
/api/search?q=query GET q / search (String, Required), page (Int), perPage (Int) ONLINE

Catalog Explorer & Advanced Filters

The Catalog Endpoint supports filtering based on various AniList metadata tags like status, genres, release format, and title strings.

Filter Name Type Allowed Values Description
genre String Action, Adventure, Comedy, Drama, Fantasy, Romance, etc. Filters media associated with the exact genre tag.
status String RELEASING, FINISHED, NOT_YET_RELEASED, CANCELLED Filters by AniList publication status.
format String TV, TV_SHORT, MOVIE, SPECIAL, OVA, ONA Filters by release format layout.

Airing Schedule Timeline Endpoint

Fetch schedules for upcoming episodes using Unix timestamps to define intervals. Required for building calendar grids or seasonal countdown lists.

Query Param Required Format Description
start Yes Unix timestamp (e.g. 1700000000) Finds airing schedules with broadcast timestamps greater than this value.
end Yes Unix timestamp (e.g. 1700086400) Finds airing schedules with broadcast timestamps lesser than this value.

API Usage Documentation

Semua endpoint di bawah adalah HTTP GET proxy ke AniList GraphQL. Response utama mengikuti shape AniList, jadi client KinoHarth bisa membaca Page.media, Page.airingSchedules, atau Media langsung dari JSON.

Base URL
https://your-api-project.vercel.app
Untuk local dev, ganti base URL dengan origin saat ini, misalnya http://localhost:3005.
Optional Auth Header
X-KinoHarth-Key: your_key_here
Header ini wajib hanya kalau environment KINOHARTH_API_KEY diset di server.
Endpoint Purpose Required Params Try
GET /api/health Check proxy status. None
GET /api/search?q=naruto Search anime title from AniList. q or search
GET /api/trending Trending anime sorted by AniList trend/popularity. None
GET /api/popular Popular anime sorted by popularity and score. None
GET /api/catalog?genre=Action Browse with filters: title, genre, status, format. None
GET /api/anime/21 or /api/21 Get detail metadata, trailer, characters, relations. Anime ID in path
GET /api/schedule?start=...&end=... Upcoming airing episodes in Unix timestamp range. start, end
curl Example 
curl "https://your-api-project.vercel.app/api/search?q=one%20piece&perPage=5" \
  -H "Accept: application/json" \
  -H "X-KinoHarth-Key: your_key_here"
Frontend Fetch Example 
const res = await fetch("/api/search?q=one%20piece&perPage=5", {
  headers: { "Accept": "application/json" }
});

const data = await res.json();
const anime = data.Page.media;
